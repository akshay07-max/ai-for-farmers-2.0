```
### 🔐 Auth (Step 1)
```
# 1. Register (watch terminal for OTP printed + real SMS to phone)

{ "name": "Ashu Patil", "phone": "9876543210", "password": "Test@1234", "district": "nashik", "primaryCrops": ["onion", "wheat"] }

POST /api/v1/auth/login
{ "phone": "9876543210", "password": "Test@1234" }
